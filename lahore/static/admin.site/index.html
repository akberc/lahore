Welcome to Lahore

OpenShift is configured to execute toplevel method "run" in a "com.dgwave.lahore.core" module. You can change module name in "ceylon.properties" file located in "myfirstapp/.openshift/config/".

When you push Ceylon sources, they are compiled by OpenShif, if you want to upload pre compiled modules, you can place them in "myfirstapp/.openshift/modules/" folder.
